# System dependencies note:
# The following tools need to be installed separately on the system:
# - GDAL/OGR (for ogr2ogr command): brew install gdal (macOS) or apt-get install gdal-bin (Ubuntu)
# - Tippecanoe (for PMTiles generation): brew install tippecanoe (macOS) or build from source
#
# Installation commands:
# macOS:
#   brew install gdal tippecanoe
#
# Ubuntu/Debian:
#   sudo apt-get update
#   sudo apt-get install gdal-bin
#   # For tippecanoe, build from source: https://github.com/felt/tippecanoe
#
# Usage:
#   pip install -r requirements.txt
#   python compile_pmn.py <process_id> <year>
#
# Example:
#   python compile_pmn.py "proc_001" 2025
